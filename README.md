This project is a simple technical documentation site covering fundamental web‑development topics: HTML, CSS, JavaScript, responsive design, and debugging tips. It’s designed to help beginners follow step‑by‑step and understand core web concepts in a clean, easy‑to‑read layout.

# Sections
- HTML Introduction — shows basic HTML tags, and explains page structure.
- CSS Styling — covers styling techniques, use of external stylesheets, inline vs internal vs external CSS.
- JavaScript Basics — introduces variables, functions, console logging and basic interactivity.
- Responsive Design — demonstrates media queries, flexbox and grid to make layouts adapt to different screen sizes.
- Debugging Tips — gives simple guidance on how to identify and fix common coding mistakes using console logs and developer tools.
